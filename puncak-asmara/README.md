## How to add real photos/videos
Every image slot in `brochure/index.html` is a labeled placeholder, e.g.
`data-label="hero.jpg — sharp, bright daytime villa+pool"`.
Replace each placeholder `<div class="ph" ...></div>` with the matching photo
(or we wire them up automatically once the files are in `assets/photos/`).
