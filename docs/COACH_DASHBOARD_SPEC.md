# Core for Coaches — B2B Spec

**One-line pitch:** Give addiction coaches a co-pilot. They see every client's daily slips, streaks, and triggers in one dashboard. AI summaries replace the hour they spend prepping for each session.

---

## ICP

**Primary:** Solo addiction/recovery coaches with 10–50 active clients
- Charge $200–600/mo per client
- Currently use Notion + Google Sheets + WhatsApp
- Time-poor, want to look more "high-tech" than their competitors

**Secondary:** Small wellness clinics, sober coaches, vape-quit specialists
**Tertiary:** Therapists offering habit-focused CBT (post-launch, regulatory caution)

## What coaches see

### Dashboard (their view)
- All clients listed with photo, Life Score, last activity time, current streak per habit
- Color-coded urgency: red = needs check-in today, yellow = drifting, green = on track
- "AI brief" panel — 5-bullet daily summary across whole client roster
- Calendar with auto-scheduled check-in slots based on client patterns

### Client deep-view
- Same drill-down as the consumer app — but coach can see everything
- Slip patterns highlighted (3pm trigger, weekend slips, etc.)
- Coach can leave notes the client never sees
- "Prep notes for next session" auto-generated by AI
- Direct DM to client inside the app (Core handles delivery)

### Coach tools
- **Cohort mode** — run a group quit-vape program with shared accountability
- **Custom protocols** — coach can override default Core defaults for any client (different XP weights, different stat formulas)
- **Resource library** — coach uploads their PDFs/videos, clients see them in-app
- **Branded experience** (Practice tier only) — client app shows coach's logo

## Differentiation vs existing tools

| Tool | What it does | Core for Coaches advantage |
|------|--------------|-----------------------------|
| Notion | Manual tracking | Live data from client's actual phone |
| Practice (the platform) | Client management for coaches | Habit-specific AI insights, not general |
| Cliniko / SimplePractice | Therapy notes + billing | Daily habit data + AI prep notes |
| Quit-vape coaching templates (Etsy) | Static templates | Live patient data + intervention timing |

**Killer feature:** AI prep notes that read like "Tom is on day 12 vape clean. He slipped Tuesday at 3pm (work stress). His Brain stat dropped 8 points this week, primarily from doom-scroll. In session, focus on: (1) 3pm replacement ritual, (2) why phone is up after 10pm." Coaches spend 0 min prep vs. their usual 30 min.

## Tech approach

- **Same backend** as consumer Core (Supabase)
- **Multi-tenancy** added via `coach_id` foreign key on user records
- **Web dashboard** (Next.js, not RN) — coaches work on laptops, not phones
- Lives at `apps/web` (already scaffolded)
- Reuse Supabase Auth + RLS — coach can only see their own clients
- Reuse `stat-engine.ts` and habit data structures verbatim
